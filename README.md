# dnd-tools

A web app for browsing and filtering D&D spells with search, sorting, and multilingual support (English / French).

## Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- TanStack Table

## Contributing

1. Fork the repo
2. Create a feature branch (`git checkout -b feat/my-feature`)
3. Install dependencies: `npm install`
4. Start the dev server: `npm run dev`
5. Make your changes, then verify the build: `npm run build`
6. Open a pull request
